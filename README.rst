Usage
-----

Haystack is simple. Beautiful. ::

    import haystack

Installation
------------

Installing haystack is easy::

    $ pip install haystack

Or, if you must::

    $ easy_install haystack

But, `you really shouldn't do that
<http://www.pip-installer.org/en/latest/#pip-compared-to-easy-install>`_.
